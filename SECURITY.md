# Security Policy

## Reporting a vulnerability

Please do **not** open a public issue for security problems. Report privately by email to the maintainer (see the repository owner's profile). Include steps to reproduce and impact. We aim to respond within a few days.

## Scope

In scope: the client cryptography and vault format, key handling, local storage, sync-to-folder logic.

## Status

This is **beta** software. The cryptography has **not** yet had an independent third-party audit. Do not store irreplaceable secrets without your own backups.
