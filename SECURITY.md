# Security Policy / Политика безопасности

## Reporting a vulnerability / Сообщить об уязвимости

**EN:** Please do **not** open a public issue for security problems. Report privately by email to the maintainer (see the repository owner's profile). Include steps to reproduce and impact. We aim to respond within a few days.

**RU:** Пожалуйста, **не** создавайте публичный issue по проблемам безопасности. Сообщайте приватно на почту мейнтейнера (см. профиль владельца репозитория). Опишите шаги воспроизведения и влияние. Постараемся ответить в течение нескольких дней.

## Scope / Область

In scope: the client cryptography and vault format, key handling, local storage, sync-to-folder logic.
В области: криптография клиента и формат хранилища, работа с ключами, локальное хранение, синхронизация в папку.

## Status / Статус

This is **beta** software. The cryptography has **not** yet had an independent third‑party audit. Do not store irreplaceable secrets without your own backups.
Это **бета**. Криптография **не** проходила независимый сторонний аудит. Не храните незаменимые секреты без собственных резервных копий.
