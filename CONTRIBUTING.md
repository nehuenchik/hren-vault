# Contributing

Thanks for your interest!

## Ground rules

By submitting a contribution you agree it is licensed under the project's license (GNU AGPL-3.0) and you sign off each commit under the Developer Certificate of Origin (DCO): commit with `git commit -s` (adds a `Signed-off-by` line).

## Before a PR

- `cargo build --release` and `cargo test` pass.
- `cargo clippy` without new warnings.
- Do not add telemetry or send user data anywhere.

## Security

For vulnerabilities see `SECURITY.md` — report privately, not via public issues.
